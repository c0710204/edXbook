**********
创建一个单元
**********

   1. 在 **课程大纲** 上，选中您想要创建单元的节。

   2. 点击 **新单元** 如下的画面会显示。

    .. image:: Images/C04_01.png
       :width: 800


   3. 在 **单元名称** 栏中，输入单元名称。这个名称将会显示在 Edge 画面上方的单元列表中。

      每个单元有一个或多个组件。

     - 创建一个论坛，您或您的学生就可以发布问题或者参与讨论，点击 **论坛**

     - 创建一个组件，您就可以增加文字、图片，或者其他内容，点击 **HTML**

     - 创建一个问题给您的学生解决，点击 **问题**

     - 新增一个视频，点击 **视频**
