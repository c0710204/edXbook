**********
创建一个单元
**********

   1. 在 **课程内容** 上，点选您想要创建单元的小节。

   2. 点击 **建立新单元** 如下的画面会显示。

    .. image:: Images/C04_01.png
       :width: 800


   3. 在 **显示名称** 栏位中，填入单元名称。在 Edge 上画面上方这个名称会显示在单元导览列中。

      每个单元有一个或多个组件。

     - 创建一个讨论空间，您或您的学生就可以发布问题或者参与讨论，点击 **讨论**

     - 创建一个组件，您就可以增加文字、图片，或者其他内容，点击 **HTML**

     - 创建一个问题给您的学生解决，点击 **问题**

     - 新增一个影片，点击 **影片**
