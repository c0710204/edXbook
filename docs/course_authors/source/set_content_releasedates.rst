*********************
设定内容发布日期及能见度
*********************

课程教材的发布时程表是由章节与小节的发布日期设定决定的。


章节
****

    要设定章节的发行日期:

    1. 在 **内容** 选单点击 **课程内容** 。

    2. 在课程内容里寻找您要找的章节。

    3. 在 **即将发布** 下方，点击 **编辑** 来改变日期。

 .. image:: Images/C09_01.png

小节
****

小节预设是继承其所在小节的发布日期，但您可以更改这个日期，如此一来不同的小节能在不同的时间发布。

注意若小节的发布日期比在其所在的章节的发布日期更早，学生在此 **章节** 发布日期前将无法看到此小节，章节的发布日期会覆写小节的发布日期。 

   要为小节设定发布日期:

    1. 点击以开启小节。

    2. 找到在右上角的 **小节设定** 方格。

    3. 在 **发布日期ㄧ** 下方修改日期。

单元
****
     
个别的单元会继承其所在的小节的发布日期，但其有一个额外的 **能见度** 设定可将其设定为 **公开** 或 **私有** ，私有单元将永不会被学生看到。

.. note::

   您能直接修改私有单元，但您必须创建一个草稿才能修改被标记为公开的单元。

观看更多讯息，见 :doc:`modify_published_content` .

    
       
    要变更私有单元的 **能见度** 设定:
  
  
    1. 点击以开启单元。
  
  
    2. 找到位於右上角的 **单元设定** 方格。
  
  
    3. 於 **能见度** ，选取 **公开** 。
  
  
    要变更公开单元的 **能见度** 设定:
  
  
    1. 点击以开启单元。
  
  
    2. 找到位於右上角的 **单元设定** 方格。
  
  
    3. 在 **单元设定** 下方，点击 **编辑草稿** 。
  
  
    4. 於 **能见度** ，选取 **私有** 。
