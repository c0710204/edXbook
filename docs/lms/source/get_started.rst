############
开始使用 LMS
############

****
简介
****

当我们开始发行 edX 给我们最初的夥伴时，我们已经提供一些机会让另外的教师们在我们的平台上创建一些课程。
我们努力的果实是 Edge 与 Studio。
这些工具不只对我们的 edX 平台有用，还能对所有大学协会的所有教职员有所帮助。

EdX (http://edx.org) 是我们独创，最初的学习入口。
在 edX 上能做的事情，端看於您的学校和 edX 达成的协议内容。
您需要您学校的特别批准以将您的课程放置到 edX 上。
一旦课程被放置到 edX 入口，它就变成一个公开大型开放式线上课程(MOOC)。

LMS 是 edX 提供给各使用者所观看的介面。
为了让使用者能够寻找欲学习的课程，并透过各课程提供的课程资讯、作业与考试，以及与其他使用者的互动，来达到学习与获取知识的目的。
使用者可以透过欲学习的课程名称或是开课单位，在 LMS 中寻找欲学习的课程注册，并经由各开课单位所提供的课程内容，作为学习的途径。

这里有个如何开始的工作流程，以下是份简易的总结:

1. 首先进入: https://www.edx.org/ 注册并且创建一个帐号。

2. 在 LMS 寻找欲学习的课程。

 :width: 800

3. 在控制台中点选已注册的课程连结，开始检索所有课程相关资讯以及各种操作。

************
创建一个帐号
************

要开始使用 LMS，您必须先以一个独一无二的使用者名称及密码创建一个帐号。
在 LMS 当中所有的学习行为与学习历程，都将依据您的帐号资讯记录在 LMS 当中，让您在未来能够检视过去的学习情形。

1. 进入 http://www.edx.org/

.. image:: Images/image001.jpg
 :width: 800

2. 点选右上方的 Login 连结，进入登入页面。

.. image:: Images/image004.jpg
 :width: 800
 
3. 点选 Sign up for edX today! 连结，进入创建帐号页面。

.. image:: Images/image005.jpg
 :width: 800

4. 填写帐号申请之相关资料，有标注星号(*)之栏位为必须填写的资料。

.. image:: Images/image006.jpg
 :width: 800

5. 完成注册，将自动跳转至个人控制台页面。

.. image:: Images/image007.jpg
 :width: 800

********
寻找课程
********

当您首次登入 LMS ，进入个人控制台页面时，您将会看到以下的画面。

.. image:: Images/image007.jpg
 :width: 800

点击 Find courses now! 连结进入「课程索引」页面，开始寻找欲学习的课程。
进入课程索引页面後，您将看到以下的画面。

.. image:: Images/image008.jpg
 :width: 800
 
在 LMS 中提供了两种途径寻找您所需要的课程，分别是「开课单位搜寻」以及「主题式搜寻」的方式，以下将为您介绍如何透过这两种途径，找到您想学习的课程。

**主题式搜寻课程**
 ----

在 LMS 当中，您可透过欲学习的主题，来搜寻您所需要的课程，以便注册参与课程学习。

主题式的搜寻方式，能让您依照个人的兴趣、专长、需求，来寻找您所需要的课程，在选择对应的主题後，即可快速地寻找到您欲学习的课程，注册该课程并检索该课程的相关内容。

课程索引页面的右上方，有两个下拉式选单，左边的下拉式选单内容即为各个主题的名称。
请点选下拉式选单中，您欲了解的主题，以检视该主题中所开设的所有课程。

.. image:: Images/image009.jpg
 :width: 800

.. image:: Images/image010.jpg
 :width: 800

**以开课单位搜寻**
 ----

另一个寻找课程的方式，是透过开课单位搜寻您欲学习的课程。
若您要找寻某个学校所开设的所有课程，即可透过此种搜寻方式，以便快速寻找到您所需要的课程注册後，能够开始检索课程的相关资讯。

课程索引页面的右上方，有两个下拉式选单，右边的下拉式选单内容即为各个学校的名称。
请点选下拉式选单中，您欲寻找课程的开课单位，以检视该开课单位所开设的所有课程。

.. image:: Images/image011.jpg
 :width: 800
 
.. image:: Images/image012.jpg
 :width: 800

另外，除了分别使用「主题式搜寻」以及「开课单位搜寻」的方式，也可以利用『关联式搜寻』的方式，同时定义主题以及开课单位，更能精准快速的找到对应的相关课程，让您能够注册欲学习的课程，并开始检索课程的相关内容。

************
注册一门课程
************

在 LMS 中的学习，都是从注册一门课程开始，紧接着您可进入这门课程页面，检索课程的相关资讯、与其他学习者共同讨论、进行作业练习以及检阅您的学习历程等。

首先，在寻找到您欲学习的课程後，点击课程清单右边的 learn more 连结或是课程图片，以进入该「课程细节」的页面中。

.. image:: Images/image013.jpg
 :width: 800

在此课程细节的页面中，有课程的开课单位、课程编号、开课时间、课程学习时间以及每周建议学习者所需投入多少时间等资讯；并且还有课程的简介、课程讲师以及常见的问答集。
您可详细阅读这些资讯，来对课程做初步的了解。

在阅读完这些相关资讯後，您可点击 Register for XXXXX (XXXXX为课程编号)之连结以注册该课程。

.. image:: Images/image014.jpg
 :width: 800

注册完成後，将自动跳转至您的控制台页面当中，同时您会发现，该课程已出现在您的课程学习清单内。

.. image:: Images/image015.jpg
 :width: 800

********
检视课程
********

一旦注册了一门课程以後，您将可从控制台页面来选择您的课程，以进入课程进行学习。
延续「注册一门课程」的介绍，您可透过以下画面，在您欲浏览的课程，点击「View Course」连结进入课程。
 
.. image:: Images/image016.jpg
 :width: 800

**课程检索**
 ----

进入课程检索页面後，您将被自动转载至这门课程的「最新消息与更新」分页，您可看见关於这门课程的最新资讯，像是开课教师近期内发布了哪些投影片、出了哪些作业与希望缴交的时间、对於近期课程的进程与介绍，以及接下来的课程活动等。
同时，您将可透过其他的分页，看到这门课程的所有相关资讯。
一门课程的分页所记载的相关资讯如下所列：

.. image:: Images/image017.jpg
 :width: 800

`Course Info`:
此分页为这门课程的新增公告或更新，考试的提醒，学生课程行事历的变动通知，以及学生需要铭记的重要步骤等，主要说明本课程目前概况。

`Courseware`:
此分页将呈现这门课程当中的投影片、影片、作业练习、考试等活动，可进入这个分页中，以进行投影片的浏览、影片的观看，以及提交您的作业与参加考试等。
同时，可观看每位学习者餐与各个活动时，所提出的问题或是意见，并与他们进行讨论。

`Calendar`:
此分页为课程详细的时程表，内容记载着每周的期间，期间内所教授的主题、章节，与该主题、章节相关联的投影片与影片，作业与考试的规划，以及建议完成此进度的截止时间。

`Wiki`:
此分页提供所有课程餐与者，可以共同创建与编修一个关於此课程的 Wiki 讨论，使用上与协作 Wiki 的方式大致相同，得以观看、编辑、修改各项内容。

`Discussion`:
此分页提供所有课程餐与者，可以开启一个新的讨论串或餐与他人的讨论。
能藉由搜寻与排序的方式，找到欲了解或询问的相关主题，亦可针对各学习者所提出的意见推荐或是回报给管理者，同时可以跟随有兴趣的任一项讨论串，在该讨论串有相关更新时，会自动通知跟随此讨论串的餐与者。
此讨论区建立给授课者与授课者间、学习者与学习者间、授课者与学习者间三种关联，提供一个询问问题与讨论的管道。

`Progress`:
进度分析表呈现了该学习者对於每项活动的完成度。
当您餐与了一门课程，不论是浏览投影片、观看影片、提交作业、餐与考试，皆会针对您的学习行为做评比与纪录，这个分页将呈现您在这门课程中的所有表现情况，以让您了解您对於这门课程的学习情形，以及是否尚有未完成的课程活动等。
 
`Textbook`:
此分页为介绍该门课程所建议使用的教课书与教材，您可透过浏览此分页，了解这门课程所使用的教材与教课书资讯，以协助您增加学习效率及对该课程更进一步的了解。

**取消注册**
 ----

如果您发现对於您已注册的课程，其实并不是您所想学习的课程，可执行取消注册的动作，来取消注册该门课程。
在学习一门课程的期间，或许会发现所学习的课程过於困难，亦或是并非有兴趣、想学习的主题，又或着因为缺少太多的课程活动，导致学习效果不佳的情况发生。
此时，您可以考虑取消注册该门课程，待下次开课时再重新注册学习，或是找寻其他您需要的课程。

请进入控制台中，在欲取消注册的课程右下角，点选「取消注册」的连结，接着点击弹跳视窗中的「取消注册」做二次的确认，即可取消该课程的注册。
该课程将从您的控制台中删除，并且清除您的学习纪录。
