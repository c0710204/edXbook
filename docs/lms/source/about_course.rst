########
关於课程
########

关於课程这个章节，将详细的向您介绍课程资讯页面中，各个分页的功能如何使用。
诚如前章所介绍，在这个课程资讯当中拥有「课程资讯」、「教材」、「课程讨论区」、「Wiki」、「学习进度」、「教科书」这些分页资讯。本章将引领您如何使用各个分页提供的功能，让您能顺利浏览课程内容，并与其他学习者、教师进行互动，以达到学习的目的。

********
课程资讯
********

此分页为这门课程的新增公告或更新，作业与考试的提醒，学生课程行事历的变动通知，以及学生需要铭记的重要步骤等，主要说明本课程目前概况。

当您从控制台点击课程连结进入课程页面的同时，系统会自动将您转载至此页面。
此页面为每一门课程的课程主页，对於课程的现况做简要的说明，以让您了解目前课程进度为何，目前进行的作业或是考试是哪些，并且可以透过提供的连结，快速地进入课程的活动以进行课程活动。

以「MITx: 6.00.1x Introduction to Computer Science and Programming」课程为例:

您可看见此页中间的部分，有对於目前本周课程的进度概述，以及建议学习者投入多少的学习时间来进行这些活动，像是作业是否批改完成，同时会提供连结供学习者点选，以让您点击後可快速进入该页面，了解相关的资讯。

************
检视教材内容
************
 
教材在 edX 的课程当中，是以「章节」的呈现方式存在。
「章节」是最顶层的类别，可以用它来组织您的课程。
许多教师根据位於课程第几周来命名，像是第一章被命名为「第一周」、第二章被命名为「第二周」，以此类推。
章当中包含了「小节」，里面包着更多「单元」。有些章节与单元，会依据开课教师的设定，随着课程的进行与时间的推移，才会显示该教材内容，让您得以浏览与检索。
 
因此，您可依照您的学习状况，来观看从过去到现在的课程内容。
举例来说，如果您对於某一周所上的课程内容有疑问，或是经由作业练习与考试後，发现某一部分的课程理解不够清楚，您可透过列表中的连结，点选过去已学习过的课程再次学习，达到复习的目的。
一般教材内容的规划，开课教师多数会提前一周发布课程内容，让您可以提前了解接下来要学习的部分。
因此，您也可透过教材内容预习课程，让未来在学习课程时能有更好的成效与帮助。 
 
您可点选左方的「章节」连结，来浏览该章节的课程投影片以及影片，或是进行该章中的各个课程活动。 
 
.. image:: Images/image021.jpg
 :width: 800
 
另外，您也可以透过上方所提供的「小节」连结，来辅助您更迅速的检视该章节中的每项课程活动，像是随堂练习、作业提交、考试，或是与其他学习者讨论该章节的课程内容。

课程大纲
********

课程大纲会列出所有课程底下所有的静态页面，这些页面能协助您快速存取课程的重要页面。
静态页面不见得以一个特定的顺序分布，端看开课教师对课程内容的规划。
举例来说，静态页面中可能包含该课程的教学大纲或该课程的评分标准，又您应该怎麽样获取这些分数，以及建议您投入课程的时间。

课程时程表
**********

在每一门课程当中，会为您提供一个课程的时程表，开课教师可以创建一个 Google 行事历让学生可以合并进他们个人时程表中，或者会使用 HTML 去创建一个动态行事历让学生可以点击连结。
您可透过课程提供的时程表，来新增您个人学习的时程表。
以下将介绍一个透过课程时程表来完成活动的方式，首先您可点击课程时程表以检视该课程的课程时程表，接着您可点选欲进行的活动，让页面转载至该活动对应的页面，以进行该课程活动。

课程讨论区
**********

您可以在您所注册的课程中，进入该课程的「课程讨论区」，与其他学习者或是教师讨论，达成沟通及互动的目的。
课程讨论区可让您提出对於课程内容的疑问，或是提供补充资料给其他学习者。
同时，您可藉由课程讨论区了解其他学习者对於课程的了解程度，像是其他学习者所发问的问题类型、相关资料，对您在学习课程时将更有帮助。
基於与其他学习者及教师的互动，可以增加个人对於该课程的了解，以及解决问题的能力，更可以透过解决其他学习者的问题来增加自己对课程的了解程度。
另外，您可以追随任何一项讨论主题，在该主题有任何更新的时候，将会发出通知提醒您前来检视。
在追随任何一个讨论主题的时候，您也可以针对其他学习者所提出的问题、回应作评论，以推荐其他学习者来观看这项主题或是回应。
接下来，将一一介绍讨论区所提供的各项功能。

.. image:: Images/image029.jpg
 :width: 800
 
**检视讨论主题**
 ----

进入课程讨论区後，讨论区的左边将列出目前拥有的各项主题、推荐次数与回应次数，您可透过点选各项主题来检视各项主题的内容，接着可再针对该主题做推荐与回应等等的讨论区互动。

.. image:: Images/image030.jpg
 :width: 800

**发表文章**
 ----

进入课程讨论区後，若您在左边的主题列当中找不到您所需要的讨论主题，您可点击 New Post 连结，以发布新的主题来请其他课程参与者来参与您的讨论。

如下图所示，您可点击 New Post 连结来发布一个全新的主题，您必须填入该主题的名称、讨论的内容，同时您可点选左边的下拉式选单，来决定您所发布的主题是与课程哪个部分相关联，让其他学习者更能容易搜寻及了解这个主题是在探讨课程中的哪些部分。
当您填写完对应栏位中的资料後，即可点击 Add post 张贴这篇讨论，或是可以点击 Cancel 取消发布。

.. image:: Images/image031.jpg
 :width: 800

**追随与推荐主题**
 ----
 
当您进入课程讨论区後，在课程讨论区左边点选您想观看的讨论主题，即可观看该主题目前讨论及回应的内容。
如果您对该主题非常有兴趣，或是这个问题尚未有其他学习者或教师回答，而您也想了解这个问题的答案，您可选择追随这项讨论主题。
而若您觉得这项主题非常有趣，或是您认为其中某项回应非常有参考价值，您可透过推荐主题的方式，来推荐给其他课程参与者观看，进而增进所有课程学习者的互动，让讨论的主题更加热烈。

您可点击讨论主题的右上角，如下图中红笔圈选的地方，来追随这项讨论主题，日後若该讨论主题有新增的回应或是新增的内容，将会通知您前来观看。
而若您要取消追随这项讨论主题，只要再次点击即可。 
 
.. image:: Images/image032.jpg
 :width: 800

同时，您亦可点击右上角的「+」号，如下图红笔圈选处，来推荐这个主题。
当您点击推荐主题後，推荐的次数将会增加一次。
在此之後，前来主题讨论区寻找主题的其他学习者，即可透过推荐次数作为参考依据，来选择是否要观看这项讨论主题。

.. image:: Images/image033.jpg
 :width: 800

**发表回应与推荐回应**
 ----

在您进入课程讨论区後，选择了讨论主题，若您对於该主题的发表内容有兴趣，同时您想发表一些您个人的意见，可透过发表回应的方式来对主题做回应，或是针对其他课程参与者的回应发表回应，并且可以推荐某篇回应给其他课程参与者做为参考的依据。

您可将页面卷轴拉到页面的最下方，会有发布回应的栏位，在栏位中填写您对这项主题想发表的意见後，按下 Submit 连结提交至讨论区。

.. image:: Images/image034.jpg
 :width: 800
 
同时，您也可针对某篇回应，发表您个人的意见与回应。
另外，若您觉得某篇回覆非常具有参考价值，您可推荐这篇回应。
点击该篇回覆右上角的「+」号，即可推荐这篇回覆给其他课程参与者。
 
.. image:: Images/image035.jpg
 :width: 800 
 
**寻找讨论串**
 ----
 
若您想要在讨论区列表中，寻找某特定关键词的讨论主题，您可透过讨论区提供的搜寻功能，来协助您找到欲观看的主题。
同时，您可透过排序的方式，来协助您观看各项主题，而 edX 提供三种排序的方式分别为：时间（由近而远）、推荐次数（由多至少）、回应次数（由多至少）。
 
您可在如下图红笔圈选处，键入您欲查询的关键词，来协助您寻找特定的讨论主题。

********
检阅历程
********

在学习的过程当中，时不时的检阅自己的学习状况，是一件非常重要的事情。
除了能确实明白自己是不是对课程的某些部分已完全了解，同时也可藉此机会补强自己缺乏的部分，亦可以做第二次的复习，以检验自己是否能学以致用。
因此， edX 在您学习的时候，记录了您学习各课程的历程，像是作业是否缴交、考试是否餐与、随堂练习是否完成，而每项活动又拿到了多少分数等。

当您点击「Progress」连结时，您会看到如下图所示的进度表。您可透过此进度表，来了解自己对於这门课程的完成度有多少，以及各个部分的完成度又有多少，以对自己的学习做一个检验。

.. image:: Images/image038.jpg
 :width: 800 

另外，您也可透过进度表下方，edX 提供列表方式的进度表，来了解各个作业与考试的分数，进而了解自己的学习情形。
同时，列表中的每个部分，都与课程章节及课程活动对应，当您点下该连结时，即可将页面转载至该课程页面，完成未完成的课程活动，或是达到复习及再次检验的目的。
 
.. image:: Images/image039.jpg
 :width: 800

教科书
******

开课教师将提供课程使用书籍资讯，若您在学习时需要教科书，您可参考此页面由教师提供的教科书资讯，以购买教科书，但 edX 并没有要求学习者一定要购买提供的书籍。
